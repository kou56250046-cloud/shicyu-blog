# 03 Five Agents スクリーンショット取得（手動）

## 概要

Five Agents XE3 を手動操作して命式を表示し、スクリーンショットをチャットに貼り付ける。
Claude が画像を読み解いて鑑定文を生成する。

自動操作（MCP Playwright / PowerShell）は Electron アプリの制限により断念。
手動でも 1 件あたり数分程度のため、個人ブログ運用には十分。

---

## 手順

### 1. Five Agents XE3 を起動

デスクトップの `Five Agents XE3.lnk` をダブルクリック。
スタートセンターが開いたら「**四柱推命(S)**」ボタンをクリック。

### 2. 生年月日・時刻・性別を入力

| 項目 | 入力内容 |
|------|---------|
| 生年月日 | 対象者の誕生日 |
| 出生時刻 | 時・分を入力 |
| 性別 | 男 / 女 を選択 |
| 流年 | 「現在」ボタンをクリックして今年に設定 |

### 3. 命式画面をキャプチャ

命式が表示されたら以下いずれかでキャプチャ：

- **Win + Shift + S** — Snipping Tool（範囲選択）
- **Alt + PrtSc** — アクティブウィンドウ全体

`public/screenshots/YYYY-MM-DD_名前.png` として保存する。

### 4. Claude Code チャットで鑑定文を生成

チャットに画像を貼り付け、以下のプロンプトを送信：

```
（スクリーンショットを貼り付け）

以下の情報で四柱推命の詳細鑑定文を生成し、
readings/YYYY-MM-DD_名前.md として保存してください。

名前: {name}
生年月日: {birthDate}
時刻: {birthTime}
性別: {gender}
続柄: {relationship}
流年: {year}年
スクリーンショット: /screenshots/YYYY-MM-DD_{name}.png
```
