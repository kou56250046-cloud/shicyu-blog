## Step 2: 月星座の計算

月は約27.3日で地球を一周し、1日あたり約13.176° 移動する。
1星座は30° なので、月は1星座に約2.25日滞在する。

### 計算手順

1. **基準点の設定**（精度が高い既知の月の位置から計算する）

```
基準日: 2000年1月1日 12:00 UTC
この時の月の黄経: 218.316° (蠍座 8.3°)
```

2. **対象日時との日数差を計算する**

```
日数差 = 対象のユリウス日 - 2451545.0（J2000.0）
```

3. **ユリウス日の計算**

```
Y = 年、M = 月、D = 日（小数含む）
M<=2 の場合: Y = Y-1、M = M+12

A = INT(Y/100)
B = 2 - A + INT(A/4)
JD = INT(365.25*(Y+4716)) + INT(30.6001*(M+1)) + D + B - 1524.5
```

4. **月の平均黄経を推定する**

```
月の黄経 ≈ 218.316° + 13.176396° × 日数差  （mod 360°）
```

5. **黄経から星座を決定する**

```
牡羊座: 0°〜29.99°
牡牛座: 30°〜59.99°
双子座: 60°〜89.99°
蟹座: 90°〜119.99°
獅子座: 120°〜149.99°
乙女座: 150°〜179.99°
天秤座: 180°〜209.99°
蠍座: 210°〜239.99°
射手座: 240°〜269.99°
山羊座: 270°〜299.99°
水瓶座: 300°〜329.99°
魚座: 330°〜359.99°
```

> この計算は±1〜2星座の誤差が生じる可能性があります。精密な月星座は
> 天文暦（エフェメリス）で確認してください。誤差が大きい場合は
> 「蟹座または獅子座の可能性があります」と断りを入れてください。

### アセンダント計算手順

1. **UTCへの変換**（日本標準時はJST=UTC+9）
   ```
   UTC時刻 = 出生時刻(JST) - 9時間
   ```

2. **グリニッジ恒星時（GAST）の計算**
   ```
   T = (JD - 2451545.0) / 36525  （ユリウス世紀）
   GMST(度) = 280.46061837 + 360.98564736629 × (JD - 2451545.0)
   GMST = GMST mod 360
   ```

3. **地方恒星時（LST）の計算**
   ```
   LST = GMST + 経度λ  （mod 360°）
   ```

4. **ASC黄経の計算**
   ```
   黄道傾斜 ε = 23.437°
   
   ASC の黄経 = atan2(-cos(LST), sin(LST)×cos(ε) + tan(φ)×sin(ε))
   ※ φ は緯度（ラジアンに変換して計算）
   ※ atan2の結果をmod 360°して黄経に変換
   ```

5. **ASC黄経から星座を特定する**（Step 2の黄経→星座変換テーブルを使用）
