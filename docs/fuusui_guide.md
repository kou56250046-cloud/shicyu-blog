## Step 1: 卦数（クアナンバー）の計算

### 計算式

```
① 生年の西暦下2桁の各桁を加算し1桁にする（これを X とする）
   例: 1985年生 → 8+5=13 → 1+3=4 → X=4

② 生年が1900〜1999年の場合:
   男性: 10 - X  （結果が5の場合は2に置換）
   女性: 5 + X を1桁に  （結果が5の場合は8に置換）

③ 生年が2000年以降の場合:
   男性: 9 - X  （結果が5の場合は2に置換）
   女性: 6 + X を1桁に  （結果が5の場合は8に置換）
```

### 計算例

| 生年 | 性別 | 下2桁和 | 計算 | 卦数 |
|------|------|---------|------|------|
| 1985 | 男 | 4 | 10-4 | 6 |
| 1985 | 女 | 4 | 5+4=9 | 9 |
| 1990 | 男 | 9 | 10-9 | 1 |
| 1997 | 男 | 8 | 10-8 | 2 |
| 2003 | 男 | 3 | 9-3 | 6 |
| 2000 | 男 | 0→9 | 9-9 | 0→9 ※0の場合は9 |

> 注意: 旧暦で2月3日（節分）以前の生まれは前年の卦数を使用する。

## Step 6: 玄空飛星盤の作成手順

### 6-1. 元運星（運盤）を中宮に入れる

元運の数字（例: 8運なら「8」）を中宮に配置。
洛書の順序でその他の8宮に飛星を配置する。

### 洛書（基本盤）の方位配置

```
4(SE) | 9(S) | 2(SW)
3(E)  | 5(中) | 7(W)
8(NE) | 1(N)  | 6(NW)
```

### 6-2. 飛星の飛び方（九宮飛泊）

中宮に入れた数字から、以下の順に9宮を巡る（洛書順）:
```
5(中) → 6(NW) → 7(W) → 8(NE) → 9(S) → 1(N) → 2(SW) → 3(E) → 4(SE)
```

### 6-3. 向星盤（水盤）の作成

「向」の方位に割り当てられる向星から中宮向星を決定。
向きが「順飛」か「逆飛」かで飛び方が変わる。

| 座向判定 | 向星の飛び方 |
|---------|------------|
| 旺山旺向（最吉） | 順飛 |
| 上山下水 | 逆飛 |
| 双星到向 | 向星は順飛、山星は向に集まる |
| 双星到座 | 山星は順飛、向星は座に集まる |

### 6-4. 山星盤（人盤）の作成

「座」の方位に割り当てられる山星から中宮山星を決定。
同様に順飛・逆飛を判定して配置。

### 主要な座向と飛星盤の組み合わせ（8運・9運代表例）

**8運・子山午向（北座・南向き）の場合:**
- 向星: 8（向に旺星が来る旺山旺向）
- 中宮向星: 8、中宮山星: 8
- 向星は南宮に8が来るため財運が良い

> 詳細な飛星盤は座向ごとに異なるため、「[元運] [座向]飛星盤」として逐一計算する。


### セクション構成

1. `## 🧭 卦数と命卦` — 卦数の計算過程・東西四命の判定
2. `## 🏠 八宅風水の吉凶方位` — 4吉方位・4凶方位の詳細
3. `## ⭐ 玄空飛星盤` — 元運・座向・飛星盤の全体図
4. `## 🏡 部屋別の風水診断` — 間取りごとの吉凶・改善アドバイス
5. `## 📅 2026年の方位吉凶` — 年盤との重ね合わせ・今年の注意点
6. `## 💡 風水改善アドバイス` — 吉方位の活用法・凶方位の対処法
7. `## 📝 総評` — 住環境全体のまとめと今後へのメッセージ
