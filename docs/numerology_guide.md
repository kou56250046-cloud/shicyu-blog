## 基本ルール: 数の還元

すべての計算において、最終的に**1〜9の1桁、または11・22・33（マスターナンバー）**になるまで桁を加算し続ける。

```
例: 29 → 2+9=11 → マスターナンバーなので11で止める
例: 28 → 2+8=10 → 1+0=1
例: 33 → マスターナンバーなので33で止める
例: 44 → 4+4=8 → 通常のナンバー8
```

## Step 1: ライフパスナンバー（Life Path Number）

人生の方向性・生きるテーマを示す最重要な数。

### 計算方法

生年月日の**年・月・日それぞれを1桁に還元してから合計**する。

```
計算式:
① 年の各桁を合計して還元 → 年数
② 月をそのまま（または還元） → 月数
③ 日をそのまま（または還元） → 日数
④ 年数 + 月数 + 日数 → 還元してライフパスナンバー
```

### 計算例

```
1997年6月17日生まれの場合:

① 年: 1+9+9+7 = 26 → 2+6 = 8
② 月: 6（そのまま）
③ 日: 1+7 = 8
④ 8+6+8 = 22 → マスターナンバー！ライフパス = 22
```

```
1985年3月15日生まれの場合:

① 年: 1+9+8+5 = 23 → 2+3 = 5
② 月: 3
③ 日: 1+5 = 6
④ 5+3+6 = 14 → 1+4 = 5 → ライフパス = 5
```

## Step 2: 誕生日ナンバー（Birthday Number）

誕生日の「日」だけを使う。その人に生まれつき備わった特別な才能・ギフト。

```
計算: 誕生日の「日」の数字を1桁に還元（マスターナンバー除く）

例: 17日生まれ → 1+7 = 8
例: 29日生まれ → 2+9 = 11 → マスターナンバー = 11
例: 5日生まれ → 5
```

---

## Step 3: アティテュードナンバー（Attitude Number）

外見上の態度・第一印象・人との接し方。「月＋日」から計算。

```
計算: (月) + (日の還元数) を還元

例: 6月17日 → 6+(1+7) = 6+8 = 14 → 1+4 = 5
例: 3月29日 → 3+(2+9) = 3+11 → 3+11=14 → 1+4 = 5
         ※ この場合の11はマスターナンバーとして扱わず、1+1=2にして計算:
            3+2=5
```

---

## Step 4: ディスティニーナンバー（Destiny Number / Expression Number）

アルファベット表記の氏名全体から算出。生涯にわたる使命・表現の方向。

### ピタゴラス式変換表（アルファベット）

```
1 = A, J, S
2 = B, K, T
3 = C, L, U
4 = D, M, V
5 = E, N, W
6 = F, O, X
7 = G, P, Y
8 = H, Q, Z
9 = I, R
```

### 計算方法

氏名のすべてのアルファベットを数値に変換し、合計して還元する。
名（ファーストネーム）と姓（ファミリーネーム）を別々に計算してから合計する方法と、
全部まとめて計算する方法があるが、本ガイドでは**全体を合計**する方式を使う。

### 計算例

```
KOHEI OHSHIRO の場合:

K(2) + O(6) + H(8) + E(5) + I(9) = 30 → 3+0 = 3
O(6) + H(8) + S(1) + H(8) + I(9) + R(9) + O(6) = 47 → 4+7 = 11 → マスター11

3 + 11 = 14 → 1+4 = 5

ディスティニーナンバー = 5
（※マスターナンバーが途中で出ても、最終計算では還元する）
```

## Step 5: ソウルナンバー（Soul Number / Heart's Desire Number）

氏名の**母音（A・E・I・O・U）のみ**から算出。内なる欲求・魂の望み。

```
KOHEI OHSHIRO の母音:
K-O-H-E-I  → O(6) + E(5) + I(9) = 20 → 2+0 = 2
O-H-S-H-I-R-O → O(6) + I(9) + O(6) = 21 → 2+1 = 3

2 + 3 = 5

ソウルナンバー = 5
```

---

## Step 6: パーソナリティナンバー（Personality Number）

氏名の**子音のみ**から算出。外見上の印象・社会的な見せ方。

```
KOHEI OHSHIRO の子音:
K-O-H-E-I  → K(2) + H(8) = 10 → 1+0 = 1
O-H-S-H-I-R-O → H(8) + S(1) + H(8) + R(9) = 26 → 2+6 = 8

1 + 8 = 9

パーソナリティナンバー = 9
```

## Step 7: ひらがな氏名からの算出

日本語の名前（ひらがな）は以下の変換表を使って数秘数に変換する。

### ひらがな変換表（五十音ベース）

```
あ行: あ=1, い=2, う=3, え=4, お=5
か行: か=2, き=3, く=4, け=5, こ=6
さ行: さ=3, し=4, す=5, せ=6, そ=7
た行: た=4, ち=5, つ=6, て=7, と=8
な行: な=5, に=6, ぬ=7, ね=8, の=9
は行: は=6, ひ=7, ふ=8, へ=9, ほ=1
ま行: ま=7, み=8, む=9, め=1, も=2
や行: や=8, ゆ=9, よ=1
ら行: ら=9, り=1, る=2, れ=3, ろ=4
わ行: わ=1, を=2, ん=3

濁音・半濁音: 清音と同じ数値
  が行=か行, ざ行=さ行, だ行=た行, ば行=は行, ぱ行=は行
  ※例: が=か=2, ば=は=6, ぱ=は=6

拗音（小文字）: 大文字と同じ数値
  ※例: きゃ=き+や=3+8=11→2, っ=つ=6, ょ=よ=1
```

### ひらがな版の計算例（総数・ソウル・パーソナリティ）

```
おおしろ こうへい の場合:

総数:
お(5)+お(5)+し(4)+ろ(4) = 18 → 1+8 = 9
こ(6)+う(3)+へ(9)+い(2) = 20 → 2+0 = 2
9 + 2 = 11 → マスターナンバー 11

母音（あいうえお行）のみのソウルナンバー:
お(5)+お(5)+ろ(4)※ら行=ら行ではなく母音... 
※注意: ひらがなの母音は「あいうえお」の行。
   あ行（あいうえお）=母音
   他は「子音＋母音」の組み合わせなので子音要素
   
ひらがなの場合、アルファベットほど明確な母音/子音分離は難しいため、
通常はひらがな総数（名前全体の合計）のみを算出し、
母音/子音分離はアルファベット版のみで行う。
```

## 2026年の個人年（Personal Year）計算

個人年は「その年に何のエネルギーが流れているか」を示す。

```
計算: 生年月日の月+日 を還元した数 ＋ 2026年の数（2+0+2+6=10→1）

例: 6月17日生まれの場合
  → 月+日: 6+(1+7) = 6+8 = 14 → 1+4 = 5
  → 2026年: 2+0+2+6 = 10 → 1+0 = 1
  → 個人年: 5+1 = 6

2026年は「6の個人年」 → 愛情・家族・責任・調和がテーマの年
```

### 個人年の意味

| 個人年 | テーマ | 2026年のキーワード |
|------|------|----------------|
| 1 | 新しいスタート・独立 | 新しい道の開拓 |
| 2 | 協力・待機・関係構築 | パートナーシップを育む |
| 3 | 表現・創造・楽しみ | アウトプットの年 |
| 4 | 基盤作り・努力・安定 | 地道な積み上げ |
| 5 | 変化・自由・冒険 | 大きな転換が訪れる |
| 6 | 家族・愛情・責任 | 人間関係が中心 |

| 8 | 達成・収穫・キャリア | 大きな成果が出る |

## 鑑定文の出力形式

以下のフロントマターとセクション構成に従って `readings/` にmdファイルを保存する。

### フロントマター

```yaml
---
title: "{name} の数秘術鑑定"
date: "YYYY-MM-DD"
person: "{name}"
birthDate: "YYYY-MM-DD"
gender: "男 | 女"
relationship: "本人 | 妻 | 夫 | 子 | 父 | 母 | その他"
type: "numerology"
tags: ["ライフパス{n}", "ディスティニー{n}", "2026年個人年{n}"]
lifePathNumber: {数字}
destinyNumber: {数字}
soulNumber: {数字}
personalityNumber: {数字}
personalYear2026: {数字}
---
```

### セクション構成

1. `## 🔢 数秘術の基本ナンバー一覧` — 全ナンバーの計算過程と結果
2. `## ⭐ ライフパスナンバー {n}` — 人生の方向性・テーマの詳細解説
3. `## 🎯 ディスティニーナンバー {n}` — 使命・表現の方向性
4. `## 💖 ソウルナンバー {n}` — 内なる欲求・魂の望み
5. `## 🎭 パーソナリティナンバー {n}` — 外見の印象・社会的な見せ方
6. `## ✨ 生まれ持った才能・ギフト（誕生日ナンバー）` — 誕生日ナンバーから読む資質
7. `## 🌗 ナンバー間の相互作用` — 各ナンバーの組み合わせと課題
8. `## 📅 2026年の個人年（Personal Year）` — 今年のエネルギーとテーマ
9. `## 📝 総評` — 数字が示す人生の使命と今後へのメッセージ
