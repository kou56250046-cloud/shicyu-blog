## ① 命式の構成

命式は**四柱（年柱・月柱・日柱・時柱）**に**天干・地支・蔵干**が配置された表。

```
       年柱    月柱    日柱    時柱
天干 │  干  │  干  │  干  │  干  │  ← 10干（甲乙丙丁戊己庚辛壬癸）
地支 │  支  │  支  │  支  │  支  │  ← 12支（子丑寅卯辰巳午未申酉戌亥）
蔵干 │ 蔵干 │ 蔵干 │ 蔵干 │ 蔵干 │  ← 地支に内包される干
```

### 各柱が示すもの

| 柱 | 時期の目安 | 表すもの |
|----|----------|---------|
| 年柱 | 〜29歳ごろ | 祖先・社会的立場・青年期 |
| 月柱 | 30〜59歳ごろ | 親・仕事・青壮年期（**格局判定の中心**） |
| 日柱 | 60歳以降 | **本人自身**（日干が日主） |
| 時柱 | 晩年 | 子・部下・老年期 |

## ② 日主の特定

**日柱の天干（日干）= 日主**。命式解読の中心軸。

- 日主がその人の「本質的な性格・エネルギー」を表す
- 日主の五行・陰陽を起点にすべての解釈を行う
- 例：日干が「壬」なら「壬水の日主」

## ③ 五行バランスと身旺・身弱

### 五行の相生・相剋

```
相生（助ける）: 木→火→土→金→水→木
相剋（弱める）: 木→土→水→火→金→木
```

### 身旺・身弱の判定

日主と**同じ五行**（比肩・劫財）や日主を**生じる五行**（印綬・偏印）が多いほど「身旺」。

| 判定 | 意味 | 特徴 |
|------|------|------|
| 身旺 | 日主のエネルギーが強い | 自力で行動・自我が強い |
| 中和 | バランスが取れている | 柔軟に対応できる |
| 身弱 | 日主のエネルギーが弱い | 周囲の影響を受けやすい |

### 判定のポイント

1. 月支の五行が日主と同気または生じるか確認（月支は特に影響大）
2. 天干・地支・蔵干に日主と同じ五行がいくつ現れるか数える
3. 相生・相剋の連鎖を把握する

## ④ 格局の判定

格局は「命式の構造から見た人生のテーマ・タイプ」。主に**月支の蔵干**をもとに判定する。

### 普通格局（内格）8種

| 格局名 | 条件 | タイプ |
|--------|------|--------|
| 建禄格 | 月支が日主と同じ五行（陽干） | 自力・独立型 |
| 羊刃格 | 月支が日主の劫財（陽干のみ） | 強烈な自己主張 |
| 食神格 | 月支の蔵干が食神 | 才能・創造型 |
| 傷官格 | 月支の蔵干が傷官 | 天才・革新型 |
| 財格（偏財/正財） | 月支の蔵干が財星 | 財運・実務型 |
| 官格（偏官/正官） | 月支の蔵干が官星 | 社会的地位・ルール型 |
| 印格（偏印/印綬） | 月支の蔵干が印星 | 学問・精神型 |

### 特別格局（外格）の代表例

| 格局名 | 条件 | 特徴 |
|--------|------|------|
| 従旺格 | 命式が比肩・劫財だらけ | 孤高・独立心 |
| 従強格 | 命式が印星だらけ | 精神性・学術 |
| 従財格 | 日主が弱く財星が強い | 財運追求 |
| 従官格 | 日主が弱く官星が強い | 奉仕・社会貢献 |
| 従殺格 | 日主が弱く偏官が強い | 規律・使命感 |
| 化格 | 干合して五行が変化 | 特殊な才能 |

## ⑤ 用神・喜神・忌神の特定

### 定義

| 用語 | 意味 |
|------|------|
| **用神** | 命式を調整する最も重要な五行。運勢の核心 |
| **喜神** | 用神を助ける五行。あると吉 |
| **忌神** | 用神を弱める五行。あると凶 |
| **仇神** | 忌神をさらに強める五行 |

### 用神の選び方（抑扶法が基本）

1. **身旺の場合** → 日主のエネルギーを発散・制する五行が用神
   - 食神・傷官（泄気）、財星、官星が用神候補
2. **身弱の場合** → 日主を助ける五行が用神
   - 比肩・劫財（同気）、印星が用神候補
3. **調候用神** → 季節（月支）による寒暖のバランスを取る五行

### 格局別の用神の傾向

| 格局 | 身旺時の用神 | 身弱時の用神 |
|------|------------|------------|
| 建禄・羊刃 | 財・官・食傷 | 比劫・印 |
| 食神・傷官 | 財（傷官の場合は財が最良） | 印・比劫 |
| 財格 | 官（財を守る）・食傷 | 比劫・印 |
| 官・殺格 | 印（官を通関）・財 | 比劫（身を強める） |
| 印格 | 財（印を制する）・官 | 比劫・印 |

## ⑧ 大運の読み方

大運は**10年ごとの運気の流れ**。月柱の干支から順行・逆行で計算。

### 大運の確認手順

1. **大運の干支**を命式表から確認（五Agents では自動計算済み）
2. **大運の天干**が用神・喜神か、忌神かを確認
3. **大運の地支**が用神・喜神か、忌神かを確認
4. 大運と命式の干支との**干合・支合・刑冲害合**を確認

### 大運の吉凶判断

| 大運の干支 | 判断 |
|----------|------|
| 用神・喜神の五行 | 吉運（10年間が追い風） |
| 忌神・仇神の五行 | 凶運（10年間が逆風） |
| 中立 | 普通運 |

### 大運のフェーズ分析

- **前半5年**は天干の影響が強い（精神・意識面）
- **後半5年**は地支の影響が強い（現実・行動面）

## ⑨ 流年（年運）の分析

流年はその年の干支が命式・大運に与える影響。

### 分析手順

1. 流年干支を確認
2. 流年天干と**命式天干・大運天干**との干合・剋関係を確認
3. 流年地支と**命式地支**との支合・三合・刑冲害合を確認
4. 流年の五行が用神か忌神かで吉凶を判断

### 流年と大運の関係

- 大運（吉）× 流年（吉）= 最良年
- 大運（吉）× 流年（凶）= 波乱あるが乗り越えられる
- 大運（凶）× 流年（吉）= 一時的好転
- 大運（凶）× 流年（凶）= 最も注意が必要な年

## ⑪ 鑑定文の作成手順

上記 ①〜⑩ の分析結果を以下の順で鑑定文に落とし込む。

### 出力構成

1. **命式の基本構成** — 四柱の干支を一覧化し、五行の分布を示す
2. **日主と五行バランス** — 日主の特性・身旺/身弱の説明
3. **格局と用神** — 格局の名称・用神・喜神・忌神を明示
4. **生まれ持った才能・強み** — 通変星・蔵干から読む資質
5. **性格・特徴・弱み** — 忌神の影響・陰陽バランス
6. **人生の大きな流れ（大運）** — 大運一覧と現在地・吉凶
7. **今年の運勢（流年）** — 流年干支との関係・テーマ
8. **転換期・注意すべき時期** — 刑冲害が発動する年
9. **相性メモ**（相性鑑定の場合のみ）
10. **総評** — 人生テーマと今後のメッセージ

### 鑑定の深さの基準

- 格局の種類を明示し、用神・喜神・忌神を特定する
- 通変星から才能・性格・対人関係を読み解く
- 大運の流れと**現在の大運**を示す
- 流年との干合・支合・刑冲害合を分析する
- 転換期（干支の節目・冲・刑が発動する年）を具体的な年で示す
- 相性欄は個人鑑定では省略可（相性鑑定時のみ記入）
