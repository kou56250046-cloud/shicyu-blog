# 鑑定入力ガイド

各占いをリクエストする際にチャットに貼り付けるテンプレート集。

---

## 四柱推命

Five Agents のスクリーンショットを画像として投下してから以下を送る。

```
四柱推命鑑定をしてください。

名前: 
生年月日: 年月日
出生時刻: HH:MM
性別: 男／女
続柄: 本人／妻／夫／子／父／母／その他
流年: 2026年
スクリーンショット: screenshots/YYYY-MM-DD_名前.png
```
